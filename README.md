# Deep Learning Project

This repository contains code and materials for the Deep Learning project.

## Dataset

Download the dataset from Google Drive:

- [DL Project Dataset](https://drive.google.com/drive/folders/14Yj8SMPvBkft1TrUVPoHSoe5n05mNBwx?usp=sharing)

After downloading, place the dataset files inside the local `data/` folder.

> Note: `data/` is listed in `.gitignore`, so dataset files are kept out of version control.
